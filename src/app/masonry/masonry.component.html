Pour que Masonry foncitonne bien, vous devrez également inclure ImagesLoaded, vous les trouvez aux adresses suivantes: https://masonry.desandro.com/ https://imagesloaded.desandro.com/ Commençons par inclure les librairies à partir de npm 
      {{codeNPM}}
Ensuite, il faut ajouter les fichiers JS au build d'Angular
Dans le fichier angular.json, rechercher la partie 
"build"
puis dans le build, 
"scripts"
 On peut maintenant ajouter les 2 scripts, celui de Masonry et celui de ImagesLoaded 
{{codeBuildScripts}}
 Ensuite, on peut déclarer dans le componant les types des objets que l'on utilisera avec les librairies (on triche un peu!) 
{{codeDeclares}}
 Pour le reste, vous pourrez suivre le Getting Started du site et regarder l'exemple de code qui à prochaine page.
