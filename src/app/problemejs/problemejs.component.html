Comme qu'il existe beaucoup plus de librairies JS que de librairies pour Angular, il peut être très utile d'intégrer une librairie JavaScript dans notre code Angular. Le problème d'inclure des libraires JavaScript, c'est qu'on doit souvant les instancier une fois que nos éléments sont chargés et affichés. Cela nous crée un problème puisqu'on récupère nos éléments à partir de requêtes HTTP et ils seront pas la suite affichés avec un ngFor. Ainsi, il n'y a aucun évènement dans le cycle de vie d'Angular qui nous permet d'instancier la librairie JS au bon moment.
Pour régler ce problème, nous utiliserons éléments ViewChild et ViewChildren.
